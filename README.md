# 💸 Bill Splitter & Tip Calculator

Удобное приложение для совместных обедов, которое помогает честно разделить счет и чаевые между друзьями.

## 🚀 [Live Demo](https://IrinaSheshko.github.io/Restaurany-Calculator/)

---

### 🌟 Особенности:
* **Расчет в реальном времени:** Разделение основной суммы счета и чаевых на каждого участника.
* **Выбор сервиса:** Возможность выбрать процент чаевых в зависимости от качества обслуживания.
* **Интеграция SweetAlert2:** Красивые анимированные уведомления для улучшения пользовательского опыта.
* **Доступность:** Атрибуты `aria-live="polite"` позволяют скринридерам сразу озвучивать результат расчета.

---

### 🛠 Стек технологий:
* **JavaScript (Vanilla):** Логика распределения сумм и работа с внешними библиотеками.
* **SweetAlert2:** Для стильных алертов и уведомлений.
* **HTML5 & CSS3:** Адаптивная верстка с использованием Google Fonts.

---
